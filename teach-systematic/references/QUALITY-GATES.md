# 体系版质量闸门

## 目录交付前

- [ ] 学习使命和最终能力标准具体、可验证。
- [ ] 课程深度和范围已由用户确认。
- [ ] 已研究权威来源，并在可获得时比较至少两个独立知识结构。
- [ ] 能力地图覆盖目标所需的基础、核心、实践、整合和迁移。
- [ ] 每个核心能力都映射到课程、练习或项目。
- [ ] 所有前置依赖都有来源：已确认掌握或更早课程教授。
- [ ] 核心能力、关键前置和迁移目标已映射到稳定 KP ID、标明目标等级，且账本没有无证据的乐观等级。
- [ ] 包含阶段项目、综合项目和掌握证据。
- [ ] 包含提取、间隔和适用时的交错安排。
- [ ] 可选、进阶和范围外内容均显式标记。
- [ ] 来源覆盖矩阵没有未说明的核心缺口。
- [ ] 用户已确认规划前提。

任一项失败时不要交付目录，先补充研究、访谈或修订。

## Lesson 交付前

- [ ] 本节只承担一个紧凑、可观察的成果。
- [ ] 已执行 readiness gate，并确认所有关键前置。
- [ ] 到期或疑似遗忘的关键前置已被短检查，而不是按旧等级直接假设。
- [ ] 清楚说明本节在能力地图中的位置。
- [ ] 新术语得到定义，没有隐含前置。
- [ ] 同时提供直觉模型和精确定义，并说明类比边界。
- [ ] 机制、因果链或推导没有关键跳步。
- [ ] 有完整正例、反例或边界案例。
- [ ] 覆盖与用户相关的常见误解。
- [ ] 包含带提示练习、独立练习和提取检查。
- [ ] 至少一个任务检查迁移而非复述。
- [ ] 关键声明有可信来源，来源局限得到说明。
- [ ] 没有自动生成 quiz。

任一项失败时不要交付 lesson，先修订。
