# RESOURCES.md Format

`RESOURCES.md` 是该主题经过整理筛选的一组可信来源。解释性内容中的知识应来自这里，而不是来自模型参数中的猜测。

## 模版

```md
# {主题} 资源

- [书籍：_The Science and Practice of Strength Training_ — Zatsiorsky & Kraemer](https://example.com)
  关于训练计划编排与适应机制的基础性著作。适用于：任何与周期化、恢复、强度区间有关的问题。
- [文章："How Much Should I Train?" — Greg Nuckols（Stronger By Science）](https://example.com)
  关于训练量基准的循证综述。适用于：每个肌群每周训练组数目标。
```

## 规则

* **只保留高可信来源。** 优先选择一手资料、受认可的专家、同行评审研究，以及管理严格的社区。如果某个资源本质上是披着教育外衣的营销内容，就不要收录。
* **为每一项添加注释。** 三个月后，一个单独的链接是没有用的。每一项都要添加一行说明：它涵盖什么内容，以及什么时候应该使用它。
* **明确列出缺口。** 如果某个任务目标所需的领域暂时没有好资源，就写一个 `## 缺口` 小节，列出目前缺少什么。这会推动后续继续搜索。
* **严格删减。** 如果某个资源后来被证明是错误的、浅显的，或者偏离任务目标，就应该移除，而不是埋在列表里。五个精准的来源，胜过三十个平庸的来源。
