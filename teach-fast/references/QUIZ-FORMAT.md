# 速成版 Quiz 格式

只有用户学习并讨论完对应 lesson，且明确表示准备接受测验后，才在 `./quiz/NNN-<dash-case-name>.md` 创建 quiz。

```md
# Quiz NNN：{课程标题}

- 课程：`./lessons/NNN-{lesson-name}.html`
- 模式：速成版
- 状态：未作答 | 已作答
- 分数：暂无 | {结果}
- 创建日期：YYYY-MM-DD

## 任务

1. {应用题、诊断题或小任务}
   - 用户答案：

## 反馈

_等待作答。_
```

## 规则

- 通常使用 3–5 个高信息量任务，不为凑数量增加重复题。
- 优先要求执行、解释选择、预测结果、修复错误或迁移到相邻场景。
- 只出选择题和判断题。
- 不设置纯粹误导用户的“小坑”；干扰项必须对应真实常见误解。
- Quiz 完成不自动等于掌握；依据表现判断应标为已学习或已掌握。
