# 速成版质量闸门

## 目录交付前

- [ ] 最终成果具体且可以验收。
- [ ] 时间预算和运行环境已确认。
- [ ] 每节课都直接推进最终成果。
- [ ] 关键前置已确认或安排在更早课程中教授。
- [ ] 路线包含整合、验证和排错，不只是制作步骤。
- [ ] 最终实战接近真实任务。
- [ ] 被跳过的内容、原因和补学时机均已列出。
- [ ] 核心步骤有高可信来源支撑。
- [ ] 用户已确认规划前提。

任一项失败时不要交付目录，先补充研究、访谈或修订。

## Lesson 交付前

- [ ] 本节只有一个紧凑、可观察的成果。
- [ ] 已执行 readiness gate，并确认所有关键前置。
- [ ] 新术语在首次出现时得到解释。
- [ ] 演示没有省略影响理解或复现的步骤。
- [ ] 每个关键步骤都解释了原因和预期结果。
- [ ] 用户有真实或等价的实践任务。
- [ ] 有明确的结果验证方法。
- [ ] 覆盖最可能的失败与排查。
- [ ] 关键事实和步骤有可信来源。
- [ ] 明确区分本节已覆盖内容与暂时省略内容。
- [ ] 没有自动生成 quiz。

任一项失败时不要交付 lesson，先修订。
